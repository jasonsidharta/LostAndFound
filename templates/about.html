About Lost & Found

Everything you need to know about using Lost & Found

What is it? How to Use Features Technologies FAQ

What is Lost & Found?

Lost & Found is a campus-wide platform that helps students report, search, and recover lost items. Whether you dropped your keys in the library or found someone's water bottle in the IC Building, this app connects you.

Every year, hundreds of items go missing on campus. Most are never reunited with their owners because there is no central place to report and search. Lost & Found solves this by giving the UTSC community a single, searchable board where lost and found items are matched*.

* Items are matched through the claims system, where users describe and verify ownership.

How to Use

  1. Create an account on the Sign Up page
  2. If you lost something, click Submit and select "I Lost Something"
  3. If you found something, click Submit and select "I Found Something"
  4. Fill in the item details: title, category, location, and a detailed description
  5. Browse the board to see if someone already posted a matching item
  6. Use the search bar and filters to narrow down results by category, type, or status
  7. Submit a claim on an item if you believe it's yours — explain how you can identify it
  8. The item poster reviews claims and can approve or reject them
  9. Once a claim is approved, the item is marked as Resolved

Features

Item Submission
Report lost or found items with a title, category, location, and detailed description. Each item gets its own page with a unique URL.
Search & Filter
Search items by keyword or filter by category (Electronics, Clothing, Keys, Books, ID/Cards, Other), type (Lost/Found), and status (Open/Claimed/Resolved).
Claims System
Submit a claim on any item by explaining how you can identify it. The item owner reviews and approves or rejects each claim.
Comments
Add comments to items for extra info, questions, or tips. Helps the community collaborate on reuniting items.
User Profiles
Each user has a profile showing their activity, badges earned, and contribution statistics.
Admin Dashboard
Administrators can view platform statistics, user activity, and category breakdowns in organized tables.

Technologies Used

  • Flask — Python web framework for routing, sessions, cookies, and templates
  • Jinja2 — Template engine with inheritance, conditionals, and loops
  • SQLite — Lightweight database for users, items, claims, and comments
  • JavaScript — DOM manipulation and form validation with event listeners
  • HTML5 — Semantic markup with tables, forms, lists, and description lists
  • CSS3 — Flexbox layout, diverse selectors, and pseudo-class styling

Frequently Asked Questions

Can anyone submit items?
Yes. Any registered user can submit both lost and found items.
How do claims work?
When you see an item that might be yours, submit a claim with a message explaining how you can identify it. The person who posted the item will review your claim and either approve or reject it.
What happens when a claim is approved?
The item status changes to Resolved. The two users can then coordinate pickup through comments.
Can I delete my post?
You can mark it as Resolved at any time. Admins can also manage items if needed.
What categories are available?
Electronics, Clothing, Keys, Books, ID/Cards, and Other. These cover the most common* types of lost items on campus.

* Based on common items reported at UTSC campus.
